## 一、容積率數字全解（理解全案的基礎）

| 數字 | 意義 | 來源 |
|------|------|------|
| **392%** | 原工業區變更商業區時的基準容積（整體開發70%×560%） | 民國80年都市計畫 |
| **678.91%** | 京華城購物中心實際興建的容積率（即120,284.39㎡） | 民國87年建造執照（一次性） |
| **560%** | 監察院糾正後訂定的法定上限 | 民國107年都市計畫 |
| **672%** | 560% + 自創容積獎勵20%（560%×20%＝112%）；接近678.91%的「恢復原狀」中間值 | 民國110年都委會783次決議 |
| **840%** | **最終實際允建容積率**：560% + 獎勵112% + 容積移轉30%（560%×30%＝168%） | 民國111年建造執照 |

### 容積組成拆解

```
基準容積          560%
自創容積獎勵  +   112%  （560% × 20%，非法，本案追訴核心）
容積移轉      +   168%  （560% × 30%，自購合法）
────────────────────
實際允建容積      840%
```

- **672%**：沈慶京「恢復原狀」的目標值，幾乎等同原建照的678.91%。這是行賄所追求的核心利益。
- **840%**：最終核定的實際容積率，加上自行購買的容積移轉30%，遠超原本120,284㎡換算值。
- 判決書的121億元不法利益鑑定，係以**20%獎勵部分（112%）**為計算基礎，容積移轉部分合法購買，不列入。

### ㈡ 收賄罪——法院認定有罪的五大關鍵理由

**基本事實**：沈慶京指示7名人頭以政治獻金名義，於109年3月24-26日共匯款**210萬元**至民眾黨專戶，換取柯文哲違背職務協助京華城案。

#### ⒈ 密會後旋即異常加速辦理——時間序列是鐵證

| 時間 | 事件 |
|------|------|
| 109年2月20日上午11時 | 柯沈在市長辦公室單獨密談約1小時 |
| 同日下午1:33 | 黃景茂秘書即登載隔日「京華城不公開晨會」 |
| 109年3月10日 | 柯文哲便當會公開裁示送都委會研議 |
| 109年3月24-26日 | 沈慶京以人頭完成210萬元匯款 |
| 109年4月15日 | 彭振聲代蓋甲章，將陳情案送都委會研議 |

法院認定：密會→晨會→裁示→收賄→核章，時間密接，足認雙方已形成「**可得推悉之默示合意**」。

#### ⒉ 明知違法卻仍決行——內部簽呈白紙黑字

都發局承辦人張懿萱109年2月24日簽呈：

### ㈢ 圖利罪——法院認定的三個核心論點

**不法利益規模**：京華城非法取得20%容積獎勵，多得18,463.2㎡樓地板面積，鑑定市值**121億545萬6,748元**。

#### ⒈ 法無授權——容積獎勵必須有法律依據

- 京華城案非都更地區，不得準用《都市更新建築容積獎勵辦法》
- 都市計畫法第24條本身無授權給予容積獎勵
- 本案係「**唯一一件都市計畫案卻引用都市更新容積獎勵項目**」（證人謝佩珊）

#### ⒉ 明知違法仍核章——連專家諮詢都白辦

- 109年10月12日專家諮詢會議明確指出適法性疑慮
- 副秘書長李得全加註：「**不符合土管條例等全臺北市一致性規範**」
- 吳順民親赴市府施壓後，李得全才放行
- 彭振聲見狀不敢決行，**刻意不代蓋甲章**，讓柯文哲親自核章

- 110年4月21日便當會：「**請都發局擔任PM**，俟京華城送件後儘速積極協助」

## 四、核心爭點與討論

### ㈠ 甲章爭議——送研議 vs 送公展

本案有兩張關鍵簽呈，核章方式**完全不同**：

| 文件 | 時間 | 核章方式 |
|------|------|---------|
| **送研議**（陳情案送都委會） | 109年4月15日 | **彭振聲代蓋柯文哲甲章**決行 |
| **送公展**（細部計畫案草案公展） | 109年11月11日 | **柯文哲親自蓋市長章**決行 |

**為何送公展彭振聲不代蓋甲章？**

依慣例，公展公文由工務副市長彭振聲決行即可。但彭振聲偏偏只蓋自己的副市長職章往上送，理由是：
> 「**市長蓋過章不能說不知道**」（彭振聲偵查筆錄）

總工程司楊智盛在法庭上也證稱：
> 「其他案件由彭振聲決行，**只有京華城案由柯文哲決行**，這是異常」

**辯護方的說法（陳佩琪，2024年9月）**：
> 「甲章是彭振聲幫柯文哲蓋的，柯不知情」

但此說法被彭振聲本人的偵查陳述正面推翻——彭振聲不只沒有幫柯蓋，還刻意讓柯親簽。

**法律層面**：甲章在法律上視同市長親自決行，使用前提是市長授權或知情，故送研議一事法律效果仍歸屬柯文哲。

### ㈡ 210萬謝的是560%還是678%？

**辯護方主張**：朱亞虎說「弄成560並沒有給您及市府帶來任何困擾」，210萬是謝560%已完成的事，非賄賂。

**法院認定**：
1. 560%是**107年**依監察院意見訂定的，距付款已過2年，為何現在才謝？
2. 付款時間（109年3月）夾在密會→裁示→付款→核章的密接序列中，顯示是**預付賄賂**
3. 朱亞虎那句話的語境是向李文宗「說明」款項來歷，非定義賄賂用途

**未解的疑問**：柯文哲在107年5月就對林欽榮說「看不懂」，林欽榮明確回覆依法無據。**560%本就是監察院背書的結果，柯文哲並無「弄成560」的功勞**，朱亞虎說法本身邏輯矛盾。

### ㈢ 210萬謝金 vs 121億利益——比例失衡問題

| 項目 | 金額 |
|------|------|
| 京華城獲得的不法容積利益 | 約 **121億元** |
| 柯文哲收受的賄款 | **210萬元**（比例：0.017%） |
| 應曉薇收受的賄款 | **5,250萬元** |
| 合計賄款 | **約5,460萬元** |

一個市長冒著毀掉政治生命與坐牢風險，收了121億利益的0.017%——邏輯上難以說通。

**可能的解釋角度**：
- 付款時（109年3月）最終利益未定，只是請柯文哲「開個門」，121億是後來才實現的
- 沈慶京對柯文哲的政治價值不限於現金（人脈、媒體、未來合作）
- 210萬可能是「首付」，非全部

**判決書的缺口**：法院沒有正面回應金額比例懸殊的問題，僅以「時間序列密接、對價合意成立」帶過。**此點可能成為上訴審的重要爭點**。

### ㈣ 京華城本可合法申請容積獎勵——犯罪的是「客製化」

臺北市土管條例提供了合法的容積獎勵管道（第79、80條及80條之1至80條之5）。

**問題不在於想超過560%**，而在於京華城自創三個**全臺無法源的獨家獎勵項目**（韌性城市、智慧城市、宜居城市），且只適用京華城這塊地，違反平等原則。

劉秀玲要求解釋為何不走土管條例，**京華城從未回答**。

**核心弔詭**：沈慶京若走正常管道，或許也能拿到部分容積獎勵——但他選擇行賄走捷徑，買的是「**量身訂做、不用照規矩來的客製化服務**」。

### ㈤ 都委會通過了，為何北檢不起訴都委會委員？

都委會783次會議有委員在會中明確提出適法性質疑，最後仍「共識決」通過。

**北檢採用「委員被蒙蔽」理論**：他們以為都發局已把關適法性，本身無獨立犯意。

委員徐國城在法庭上說：
> 「適法性部分不應該由都委會決定，我認為都委會沒有審查適法性的責任。」

**但這裡存在邏輯缺口**：
- 有委員在會中提出違法質疑卻仍投票通過，是被蒙蔽還是被壓制？
- 若委員真的被蒙蔽，都委會決議本身就有瑕疵，理應撤銷——北市府沒撤，北檢也沒追
- 彭振聲「球員兼裁判」擔任PM，本身就是壓制委員的機制

**此點在辯護方上訴策略上有利用空間**。

### ㈥ 林欽榮的自我矛盾——亞灣2.0「事後補法」事件

林欽榮以**檢方證人**身份於115年5月15日出庭，當庭強調：
> 「如果不是根據土管自治條例之獎勵，**多給1平方公尺就是圖利**」

柯文哲律師立即反擊：
> 「你在高雄亞灣2.0，也是用細部計畫給容積獎勵——你任內可以，到了北市彭振聲就不行？」

**亞灣2.0的問題**：高雄市施行細則明定企業營運總部容積獎勵上限為**5%**，但林欽榮主責的亞灣2.0給了**20%**，整整是上限的**4倍**。

**最關鍵的動作——「事後補法」**：林欽榮出庭被打臉後，高雄市政府於**115年5月26日**（出庭後11天）公告即將修正施行細則，新增「都市計畫於擬定或變更時，得於都市計畫書中訂定容積獎勵項目」。

**「事後補法」的意涵**：如果原本有法源，不需要補；11天內補法等同間接承認當初法源確實不足。

**法院在本案中已就亞灣2.0作出區別**（亞灣屬都更地區，有都更條例授權），但「事後補法」這個時機本身是判決書所未處理的，**可能成為上訴審重要攻防焦點**。

### ㈦ 商三特 vs 商三——「特」字的意義

**商三特（第三種商業區特別）**是專為京華城這塊地訂定的特殊分區。107年修訂後，容積率被明確訂為**560%**，與一般商三相同。

沈慶京可能的論點：「特」字應有實質意義，不能跟通案商三一樣，否則何必特別劃設？

法院立場：「特」在80年計畫書中的意義是容納原已申請執照的120,284㎡，這個保障**隨建照履行而耗盡**，並非永久綁在土地上。

**此問題在都市計畫法層面仍有爭議空間，可能是上訴審技術性論點之一。**
