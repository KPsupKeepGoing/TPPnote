# 北院新聞稿特有之重點（判決書查無對應）

> 說明：北院新聞稿為法院官方摘要（約6,000字），判決書全文逾30萬字。
> 由於新聞稿係法院自行依據判決書所撰之摘要，故新聞稿絕大多數內容在判決書中均有對應原文。
> **本文件僅列出新聞稿特有、但判決書全文查無對應之重點。**
> 反向差異（判決書有、新聞稿無）請參見判決書全文，此處不予列出。

## 一、沈慶京「面露滿意微笑離去」

### 新聞稿原文（節錄）

> 「按貪污治罪條例之收受賄賂罪……前金、後謝、分期處理（以上都包含財物與不正利益搭混付給），法律評價無異。故不論係依109年2月20日上午10時許，朱亞虎陪同沈慶京前往臺北市政府與柯文哲會面之具體過程觀察，當時沈慶京單獨進入市長辦公室與柯文哲會談約1小時，朱亞虎則在會客室外等候，未參與實際會談內容，是以雙方談話之具體細節固無從直接知悉。**然會談結束後，沈慶京面露滿意微笑離去，並由朱亞虎陪同離開，其外在反應已可反映會談結果對其具有正面意義。**復參以其後市府即啟動將京華城相關陳情案送交都市計畫委員會研議之程序，並逐步朝有利於沈慶京之方向推動，顯示會談結果已轉化為具體行政作為，可推知其等間已就違背職務行為與利益給付形成可得推悉之默示對價合意，並進而達成期約賄賂之犯意聯絡。」

### 判決書對應情形

搜尋判決書全文（逾30萬字），「面露微笑」、「面露滿意微笑」等表述**均查無對應原文**。

判決書認定109年2月20日密會後形成默示對價合意，採用的推論依據為：

| 判決書採用之推論依據 | 說明 |
|----------------|------|
| 密接時序 | 密會翌日（2月21日）即安排「不公開晨會」；3月10日柯文哲裁示送都委會研議 |
| 210萬匯款時序 | 3月24–26日匯款，夾於3月10日裁示與4月15日核章之間 |

## 二、「前金後謝」原則的援引脈絡差異

### 新聞稿用法：作為接引「微笑」推論之橋段

新聞稿的論述結構如下：

1. **法律原則宣示：** 「前金、後謝、分期處理……法律評價無異。」（即不論賄賂何時給付，犯罪均成立）
2. **接引密會過程：** 「故不論係依109年2月20日上午10時許……」
3. **提出微笑觀察：** 「然會談結束後，沈慶京面露滿意微笑離去……其外在反應已可反映會談結果對其具有正面意義。」
4. **得出推論：** 「可推知其等間已就違背職務行為與利益給付形成可得推悉之默示對價合意。」

**新聞稿中，「前金後謝」原則是論述鏈的起點，最終以「微笑離去」作為推論默示合意之直接依據。**

### 判決書用法：作為獨立法律引述

判決書（判決書全文-1.md 第832行）的用法如下：

> 「此對價與職務關係之聯結是否存在……方式為『前金』或『後謝』，皆不影響上揭犯罪之成立，**最高法院112年度台上字第1300號判決意旨可資參照**。」

**判決書中，「前金後謝」原則係援引最高法院判決作為獨立法律命題，後文不接任何關於微笑的論述，與新聞稿之論述結構截然不同。**

### 差異摘要

| 比較項目 | 北院新聞稿 | 判決書全文 |
|---------|-----------|-----------|
| 「前金後謝」定位 | 論述橋段，引出微笑觀察 | 獨立法律命題，引最高法院先例 |
| 微笑觀察 | 明確列為推論默示合意之依據 | **查無此段文字** |
| 推論依據 | 微笑＋其後行政作為啟動 | 密接時序＋簡訊往來＋匯款時序 |

## 三、各方對「微笑消失」之回應

### 鄭深元律師（柯文哲辯護人）Threads 發文

鄭深元律師於宣判後在 Threads 上發文（[@kokc4x](https://www.threads.com/@kokc4x/post/DWuBCY2kx8b)），就新聞稿與判決書之差異提出推測：

> 北院新聞稿記載沈慶京「面露滿意微笑離去」，但判決書全文中此句查無對應。鄭律師**推測**係法院在宣判後重修判決內文時，將「微笑」相關描述刪除或調整，導致新聞稿與最終判決書產生落差。

**北院否認：** 台北地院對此否認到底，強調新聞稿與判決書之間並無矛盾，拒絕承認曾於宣判後修改判決內文。

### 問題所在

此爭議出現「打臉」困境：

| 時序 | 說明 |
|------|------|
| 宣判當日（3/26） | 北院新聞稿記載「面露滿意微笑離去」 |
| 鄭律師推測 | 法院宣判後重修判決書，刪除微笑字眼 |
| **問題** | **判決書係在宣判後方正式公布——若判決書在後，則判決書才是最終版本，鄭律師的推測反而指向法院「補正」宣判時的口頭理由，而非「刪除」** |

> 換言之：若鄭律師的邏輯是「新聞稿有、判決書沒有 → 法院刪了」，則法院可反稱判決書才是準確版本，新聞稿摘要係工作人員事先準備之草稿。鄭律師需進一步論證：**新聞稿的措辭為何較判決書可信**，或法院何以無法在宣判後出具完整理由。

**北院的前例：** 北院在林俊言赴醫院訊問柯文哲一事上，同樣採取「否認到底」策略（先否認、後經媒體查證確有其事），顯示此類否認具有系統性。
