## 一、沈慶京「面露滿意微笑離去」

### 新聞稿原文（節錄）

> 「按貪污治罪條例之收受賄賂罪……前金、後謝、分期處理（以上都包含財物與不正利益搭混付給），法律評價無異。故不論係依109年2月20日上午10時許，朱亞虎陪同沈慶京前往臺北市政府與柯文哲會面之具體過程觀察，當時沈慶京單獨進入市長辦公室與柯文哲會談約1小時，朱亞虎則在會客室外等候，未參與實際會談內容，是以雙方談話之具體細節固無從直接知悉。**然會談結束後，沈慶京面露滿意微笑離去，並由朱亞虎陪同離開，其外在反應已可反映會談結果對其具有正面意義。**復參以其後市府即啟動將京華城相關陳情案送交都市計畫委員會研議之程序，並逐步朝有利於沈慶京之方向推動，顯示會談結果已轉化為具體行政作為，可推知其等間已就違背職務行為與利益給付形成可得推悉之默示對價合意，並進而達成期約賄賂之犯意聯絡。」

### 判決書對應情形

搜尋判決書全文（逾30萬字），「面露微笑」、「面露滿意微笑」等表述**均查無對應原文**。

判決書認定109年2月20日密會後形成默示對價合意，採用的推論依據為：

| 判決書採用之推論依據 | 說明 |
|----------------|------|
| 密接時序 | 密會翌日（2月21日）即安排「不公開晨會」；3月10日柯文哲裁示送都委會研議 |
| 210萬匯款時序 | 3月24–26日匯款，夾於3月10日裁示與4月15日核章之間 |
| 朱亞虎4月1日簡訊 | 「弄成560並沒有給您及市府帶來任何困擾」，附上7人捐款名單 |
| 李文宗4月5日回覆 | 「市長和我們都心存感激」，認定係轉知柯文哲後代為致謝 |

**判決書全文對於2月20日密會結束後的描述，無任何涉及沈慶京表情或外在反應之文字。**

## 二、「前金後謝」原則的援引脈絡差異

### 新聞稿用法：作為接引「微笑」推論之橋段

新聞稿的論述結構如下：

1. **法律原則宣示：** 「前金、後謝、分期處理……法律評價無異。」（即不論賄賂何時給付，犯罪均成立）
2. **接引密會過程：** 「故不論係依109年2月20日上午10時許……」
3. **提出微笑觀察：** 「然會談結束後，沈慶京面露滿意微笑離去……其外在反應已可反映會談結果對其具有正面意義。」
4. **得出推論：** 「可推知其等間已就違背職務行為與利益給付形成可得推悉之默示對價合意。」

**新聞稿中，「前金後謝」原則是論述鏈的起點，最終以「微笑離去」作為推論默示合意之直接依據。**

### 判決書用法：作為獨立法律引述

判決書（判決書全文-1.md 第832行）的用法如下：

> 「此對價與職務關係之聯結是否存在……方式為『前金』或『後謝』，皆不影響上揭犯罪之成立，**最高法院112年度台上字第1300號判決意旨可資參照**。」

**判決書中，「前金後謝」原則係援引最高法院判決作為獨立法律命題，後文不接任何關於微笑的論述，與新聞稿之論述結構截然不同。**

### 差異摘要

| 比較項目 | 北院新聞稿 | 判決書全文 |
|---------|-----------|-----------|
| 「前金後謝」定位 | 論述橋段，引出微笑觀察 | 獨立法律命題，引最高法院先例 |
| 微笑觀察 | 明確列為推論默示合意之依據 | **查無此段文字** |
| 推論依據 | 微笑＋其後行政作為啟動 | 密接時序＋簡訊往來＋匯款時序 |

## 三、各方對「微笑消失」之回應

### 公眾質疑與法律追問

> **到底是新聞稿造假還是判決書造假？請台北地方法院出來回應！**

「面露滿意微笑離去」在法院官方新聞稿中明確存在，被用來強化默示對價合意、心照不宣的賄賂約定；但在正式完整判決書中卻**完全消失**。二者均為法院官方文件，不可能同時為真。

若記載不實，將涉及：

> **中華民國刑法第214條**
> 明知為不實之事項，而使公務員登載於職務上所掌之公文書，足以生損害於公眾或他人者，處三年以下有期徒刑、拘役或一萬五千元以下罰金。

### 鄭深元律師（柯文哲辯護人）Threads 完整發文

鄭深元律師於宣判後在 Threads 發文（[@kokc4x](https://www.threads.com/@kokc4x/post/DWuBCY2kx8b)），完整原文如下：

> 一般是先寫好判決，再由判決裡面精簡一些標題及理由放到新聞稿內，不管新聞稿是受命或庭長撰寫的，或者是由行政庭長本人修改的，不可能為了新聞稿易懂，而去更動判決內文的字眼，時間上也不允許。但這件新聞稿確實有「面露滿意微笑離去」，但判決出來之後，不僅檢索不到「面露滿意微笑離去」，而且連「微笑」二個字都不見了，但是「笑容」還在。所以，我們揣摩一下判決書的論理方式，大膽推測一下，應該是宣判之後，新聞稿被引為笑柄，因此重修了判決內文，以避免失焦，是不是這樣，大家可以自行評斷，沒有對錯的問題，畢竟主文不會變（哈）。至於更動內文的人，是不是受命法官，則不得而知，畢竟從三月二十六宣判，到上傳書記官製作正本四月二日之間，有幾天的時間。這事本來沒什麼，但北院否認到底就比較令我詫異了。畢竟先前北檢也二次否認林俊言是去醫院訊問，說是去視察，非僅史無前例，而且後來發現林俊言去之前發了一個函給台北醫院說他當天要去「訊問」，請台北醫院安排場地，就不攻自破了。

**核心推論：** 鄭律師推測係宣判後新聞稿「被引為笑柄」，因而**重修判決內文**以避免失焦，時間窗口為 3/26 宣判至 4/2 書記官製作正本之間。

### 北院否認

台北地院對此否認到底，強調新聞稿與判決書之間並無矛盾，拒絕承認曾於宣判後修改判決內文。

---

### 問題所在：時序反打臉

此爭議出現邏輯困境——**判決書在後，反而成為鄭律師理論的難題**：

| 時序 | 說明 |
|------|------|
| 3/26 宣判當日 | 北院新聞稿記載「面露滿意微笑離去」 |
| 3/26–4/2 之間 | 鄭律師推測：重修判決內文，刪除微笑字眼 |
| 4/2 | 書記官製作判決書正本，判決書「笑容」在、「微笑」不在 |
| **問題** | **若判決書（4/2）在新聞稿（3/26）之後，法院可反稱：判決書才是最終準確版本，新聞稿係宣判前預備草稿。鄭律師需進一步論證：新聞稿的措辭為何較判決書可信，或法院何以在宣判後仍可修改已定讞文字。** |

> 換言之：鄭律師若主張「法院刪了微笑」，則必須解釋——**刪的是判決書，還是新聞稿本就有誤？** 若是前者，此為司法文書造假；若是後者，責任在新聞稿撰寫人員。北院的「否認到底」策略使此問題懸而未決。

**北院否認前例：** 北院在林俊言赴醫院訊問柯文哲一事上，同樣採取「否認到底」策略——先否認、後經媒體查證確有其事（林俊言出發前曾發函台北醫院要求安排「訊問」場地），顯示此類否認具有系統性。
