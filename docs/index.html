柯文哲京華城案判決書全文與辯護資料

臺灣臺北地方法院113年度金訴字第51號刑事判決。柯文哲被訴圖利罪、收賄罪。本站收錄判決書全文（逾30萬字）、起訴書全文、法庭言詞辯論逐字稿15場、北院新聞稿、ExcelPay工作簿金流爭點分析、辯方28項爭點整理、新聞稿與判決書差異、起訴書與判決書差異、敵性證人（林欽榮偽證、林洲民、蘇麗瓊、苗博雅）、黨檢媒一體造謠存證、升官發財請走此道、柯文哲自清庭訊紀錄、澄清影片87支。

關鍵爭點：某時地1500萬現金不成立、ExcelPay政治獻金非賄款、橘子許芷瑜看牙齒未限制出境、林欽榮作偽證亞灣補法、林俊言不正訊問偽造筆錄、彭振聲供詞遭脅迫、沈慶京微笑消失新聞稿差異、沈慶京不咬柯P公司就沒了、製圖小草被羈押、呂政燁綠色法官。
