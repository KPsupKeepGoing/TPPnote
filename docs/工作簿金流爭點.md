# ExcelPay 金流爭點

> ## 核心事實
>
> **柯文哲收受的是政治獻金，不是賄款。**
>
> - 沒有私用，**沒有一塊錢流進他個人口袋**
> - 政治獻金帳目記錄在工作簿中，用於黨務與選舉
> - **非法收受政治獻金屬行政違規，最高只是罰款**——不是刑事犯罪
> - 其他政黨長期同樣收受企業政治獻金，**從未被刑事追訴**，差別只在於有沒有被抓

> 臺灣臺北地方法院 113年度金訴字第51號
>
> 本文依法庭逐字稿（[12/15下午言詞辯論庭逐字稿](逐字稿2.md)、[12/16上午言詞辯論庭逐字稿](逐字稿3.md)、[12/17上午言詞辯論庭逐字稿](逐字稿4.md)、[12/17下午言詞辯論庭逐字稿](逐字稿5.md)、[12/18上午言詞辯論庭逐字稿](逐字稿6.md)）整理，對照檢察官論告與辯護人答辯，就扣案AE37行動硬碟「工作簿」Excel表**無法確認或有爭議之金流條目**進行逐一分析。
>
> **已有直接金流確認的條目**（范永偉300、黎明群500、陳盈助300、謝國樑200、林敏雄200、周俊吉200）詳見第一節檢察官驗證表，不在各條目分析中重複列出。
>
> **判決書結論：** 工作簿性質認定為傳聞證據，不符合業務紀錄傳聞例外（刑事訴訟法第159條之4第2款），**不採為直接證據**；1,500萬現金交付部分**不另諭知無罪**（即不成立）。

## 一、工作簿整體性質爭點

### 檢察官說法（[12/16上午言詞辯論庭逐字稿](逐字稿3.md)、[12/15下午言詞辯論庭逐字稿](逐字稿2.md)）

| 項目 | 內容 |
|------|------|
| 發現時間 | 113年8月30日搜索柯文哲住宅，當天下午在AE37行動硬碟「2024→財務→工作簿」路徑下發現 |
| 檔案作者 | 顯示為「WEN」；透過其他硬碟同名文件比對，確認「WEN」即柯文哲本人 |
| 最後修改日期 | 111年12月27日（柯文哲卸任市長隔二天）|
| 備份日期 | 112年2月23日；AE37硬碟上貼有「20230223 備份」黃色便條，字跡為柯文哲親筆 |
| 記帳邏輯 | 以「月」為單位記帳（10月1日群、11月1日群）；用途欄空白者皆為現金收款；有金額=有收、空白=沒給（黃崇仁例） |
| 帳冊定性 | 柯文哲親自製作以記錄誰有「幫助過我們」的私人帳冊，內容可逐筆驗證 |
| 沈慶京連結 | 111年9月1日沈慶京以「小沈」自稱邀柯文哲出席動土典禮；10月19日剪綵；其後11月1日欄出現「小沈 1500 沈慶京」 |

#### 檢察官「帳冊即金流」核心論述（[12/16上午言詞辯論庭逐字稿](逐字稿3.md)）

> 「**現金的金流要怎麼證明？難道花掉了就說沒有金流嗎？因為花掉了，我們可能扣不到這一筆現金嘛。但是我們找到了帳冊——帳冊本身就是證明現金金流的證據。**」

檢察官指出，工作簿多筆現金條目均已透過**證人到庭確認**（以下為已確認條目，不再重複於各節分析）：

| 條目 | 金流驗證方式 |
|------|-------------|
| 范永偉 300萬 | **本人承認**有送現金給柯文哲 |
| 黎明群 500萬 | **本人承認**有送現金給柯文哲 |
| 陳盈助 300萬 | 邱清章到庭確認親自到市長室交付300萬現金紙袋（有名片佐證） |
| 謝國樑 200萬 | 本人確認（母親林曼麗出資，由謝國樑轉交邱佩琳，邱佩琳在市長室當面交柯文哲） |
| 林敏雄 200萬 | 本人確認（秘書鄭和隆在東方文華交200萬現金給邱佩琳） |
| 周俊吉 200萬 | 妻子周王美文確認從家中拿200萬現金交給邱佩琳 |

> **檢察官推論：** 「同一本帳冊上，范永偉跟黎明群承認有送錢（300萬、500萬），但同樣是11月1日的這筆1,500萬，沈慶京卻不承認——同一本帳冊的現金記帳本質應做**一致的認定**。」

#### 小沈1500的具體金流鏈（檢察官版本）

1. **資金來源：** 沈慶京的秘書吳彩仙，於111年9月20日至10月17日，從永豐銀行密集提領**1,600萬元**現金
2. **時序：** 10月19日動土典禮（柯/沈同台）→ 10月25日京華城發放容積獎勵相關文件 → **11月1日工作簿出現「小沈1500」**
3. **250萬缺口：** 辯方稱其中250萬是另案交付徐文新，但「徐文新說沈慶京的水缸太大，搞不清楚錢從哪裡提領」——片面陳述，無借據佐證，無法做有利被告認定
4. **「威京小沈已給過」：** 112年11月3日柯文哲傳訊黃珊珊，除210萬外沈慶京與民眾黨無其他往來，故「已給過」即指此1,500萬
5. **帳冊一致性：** 與范永偉、黎明群等已確認條目同在同一本帳冊，應統一認定

#### 許芷瑜的角色（現金管理人）

- 蘇進強偵查中說：「許芷瑜在幫柯文哲管理私人的錢」
- 柯文哲曾傳訊蘇進強：「有需要錢的時候就直接向許芷瑜拿，現在只能付現金給你們」
- 與1,500萬收賄時間重疊（111年10月）：許芷瑜同期間收取「金張」、「曾金」共300萬；112年3月收取妙天1,000萬
- 以上顯示許芷瑜是長期負責柯文哲現金收付的實際執行人

| 項目 | 內容 |
|------|------|
| 硬碟出處 | 卷內有四種說法：主臥室床頭櫃、書桌內、電腦旁、辦公室，出處不明 |
| 取證程序 | 搜索當日未當場封緘錄影；8月30日扣押至9月11日送鑑識機關製作鏡像，中間有11天空窗，無法排除資料被植入可能 |
| 封籤狀態 | 硬碟送進法庭時「裸露」，完全無任何封籤，無法確認證據同一性 |
| 存取日期異常 | 檢察官「才一讀取就發現」，但檔案存取日期不應停在最初，有篡改疑慮 |
| 記帳日期矛盾 | 工作簿所有條目集中記載「10月1日」或「11月1日」，但無任何證人的捐款時間是這兩天，顯示製作者係依自己邏輯整理，非依事實即時記載 |
| 製作人不明 | 檢察官未找到真正的製作人加以查證，無從確保內容真實性 |
| 搜索票疑義 | 搜索票申請時依圖利罪，1,500萬收賄係「搜索後才增加的罪名」（「騙票」搜索的指控） |
| 整體錯誤率 | 多筆條目當事人否認：「Excel表裡面每一個欄位都有錯誤」 |

#### 辯方對「帳冊即金流」的反駁

| 檢察官論點 | 辯方回應 |
|-----------|---------|
| 帳冊本身就是現金金流的證明 | 帳冊是傳聞，製作人不明，不能自我證明；日期集中於10月1日/11月1日，顯非即時記帳 |
| 多筆條目有人承認→應一致認定 | 有人承認不等於每筆都真；林志朗、蔡明興等人明確否認；承認的條目不能拉高其他條目的可信度 |
| 沈慶京提領1,600萬可覆蓋1,500萬 | 沈慶京平均每月提領1,000萬是正常商業往來，111年12月還有回存1,000萬；檢察官只截取一段時間拼湊，未呈現完整金流全貌 |
| 250萬缺口（徐文新）片面陳述 | 辯方：正因「片面陳述無佐證」，應依無罪推定原則做有利被告之認定，而非反過來推定1,500萬為真 |
| 許芷瑜收現金可印證柯文哲收賄習慣 | 許芷瑜收取其他款項，不能直接跨案推論「小沈1500」也有收取 |
| 「威京小沈已給過」=確認1,500萬 | 此訊息發生於112年（2023年），比指控行為（111年）晚一年以上，屬事後訊息，且「已給過」可能僅指210萬 |

| 金流來源 | 沈慶京秘書吳彩仙在111年9月20日至10月17日，從永豐銀行密集提領1,600萬現金；扣除250萬（徐文新另案取走），餘額可覆蓋1,500萬 | 辯方：沈慶京平均每月提領1,000萬是正常商務往來；111年12月還有回存1,000萬；檢察官只截取其中一段拼湊「犯罪金流」 |

## 四、判決書最終處理

| 項目 | 判決書認定 |
|------|-----------|
| 工作簿證據能力 | 不構成業務紀錄傳聞例外（刑事訴訟法第159條之4第2款），**不採為直接證據** |
| 1,500萬現金交付 | 因無法確認交付之時間、地點、方式，亦無直接證人或物證，**不另諭知無罪**（即此項不成立） |
| 210萬匯款 | 另案採認（以時序、朱亞虎簡訊等間接證據認定，與工作簿無關） |
| 「威京小沈已給過」簡訊 | 作為柯文哲「知悉沈慶京有捐款」的**補強間接證據**，而非直接認定1,500萬的依據 |
