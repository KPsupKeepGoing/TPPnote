# 工作簿（AE37 Excel表）金流爭點：檢辯說法對照

> 臺灣臺北地方法院 113年度金訴字第51號
>
> 本文依法庭逐字稿（逐字稿2、3、4、5、6）整理，對照檢察官論告與辯護人答辯，就扣案AE37行動硬碟中「工作簿」Excel表各條目之金流說法進行逐一比較。
>
> **判決書結論：** 工作簿性質認定為傳聞證據，不符合業務紀錄傳聞例外（刑事訴訟法第159條之4第2款），**不採為直接證據**；1,500萬現金交付部分**不另諭知無罪**（即不成立）。

## 一、工作簿整體性質爭點

### 檢察官說法（逐字稿3、逐字稿2）

| 項目 | 內容 |
|------|------|
| 發現時間 | 113年8月30日搜索柯文哲住宅，當天下午在AE37行動硬碟「2024→財務→工作簿」路徑下發現 |
| 檔案作者 | 顯示為「WEN」；透過其他硬碟同名文件比對，確認「WEN」即柯文哲本人 |
| 最後修改日期 | 111年12月27日（柯文哲卸任市長隔二天）|
| 備份日期 | 112年2月23日；AE37硬碟上貼有「20230223 備份」黃色便條，字跡為柯文哲親筆 |
| 記帳邏輯 | 以「月」為單位記帳（10月1日群、11月1日群）；用途欄空白者皆為現金收款 |
| 帳冊定性 | 柯文哲親自製作以記錄誰有「幫助過我們」的私人帳冊，內容可逐筆驗證 |
| 沈慶京連結 | 111年9月1日沈慶京以「小沈」自稱邀柯文哲出席動土典禮；10月19日剪綵；其後11月1日欄出現「小沈 1500 沈慶京」 |

### 辯護人說法（逐字稿4、5、6）

| 項目 | 內容 |
|------|------|
| 硬碟出處 | 卷內有四種說法：主臥室床頭櫃、書桌內、電腦旁、辦公室，出處不明 |
| 取證程序 | 搜索當日未當場封緘錄影；8月30日扣押至9月11日送鑑識機關製作鏡像，中間有11天空窗，無法排除資料被植入可能 |
| 封籤狀態 | 硬碟送進法庭時「裸露」，完全無任何封籤，無法確認證據同一性 |
| 存取日期異常 | 檢察官「才一讀取就發現」，但檔案存取日期不應停在最初，有篡改疑慮 |
| 記帳日期矛盾 | 工作簿所有條目集中記載「10月1日」或「11月1日」，但無任何證人的捐款時間是這兩天，顯示製作者係依自己邏輯整理，非依事實即時記載 |
| 製作人不明 | 檢察官未找到真正的製作人加以查證，無從確保內容真實性 |
| 搜索票疑義 | 搜索票申請時依圖利罪，1,500萬收賄係「搜索後才增加的罪名」（「騙票」搜索的指控） |
| 整體錯誤率 | 多筆條目當事人否認：「Excel表裡面每一個欄位都有錯誤」 |

## 二、各條目逐一比較

> ✅ 检方說法中「本人確認」=有捐款的直接或間接佐證；❌「本人否認」=該人否認捐款或相關聯繫

### 條目 17：【核心條目】小沈 1500 沈慶京（1,500萬）

**工作簿記載：** 小沈 1500 沈慶京（111年11月1日群）

| | 檢察官說法 | 辯護人說法 |
|--|-----------|-----------|
| 「小沈」連結 | 111年9月1日沈慶京曾傳訊給柯文哲以「小沈」自稱，邀柯文哲出席京華城動土典禮，工作簿用語與其一致 | — |
| 金流來源 | 沈慶京秘書吳彩仙在111年9月20日至10月17日，從永豐銀行密集提領1,600萬現金；扣除250萬（徐文新另案取走），餘額可覆蓋1,500萬 | 辯方：沈慶京平均每月提領1,000萬是正常商務往來；111年12月還有回存1,000萬；檢察官只截取其中一段拼湊「犯罪金流」 |
| 交付時間 | 111年10月19日（動土典禮）至111年11月1日之間的「某時某地」 | 辯方：「某時某地」無法確認；10月19日動土典禮現場「維安人員非常多，不可能交付1,500萬」；這段時間柯文哲與沈慶京僅於動土典禮公開場合重疊一次 |
| 交付方式 | 現金直交；因缺乏金流紀錄，工作簿帳冊本身作為「現金金流的證明」 | 辯方：帳冊是傳聞，本身不能直接證明金流；同一帳冊上多筆記載有誤，小沈1500亦不能單獨採信 |
| 「威京小沈已給過」 | 112年11月3日柯文哲傳訊黃珊珊「威京小沈已給過，不要再找他」；黃珊珊理解為沈慶京已有捐款；除210萬外，沈慶京與民眾黨無其他往來，故「已給過」指1,500萬 | 辯方：「已給過」時間是在112年（2023年），此時京華城圖利行為已發生逾3年，不能以事後訊息反推109年的主觀犯意 |
| **判決書結論** | **工作簿不構成業務紀錄傳聞例外，不採為直接證據；1,500萬現金交付部分不另諭知無罪（即不成立）** | — |
