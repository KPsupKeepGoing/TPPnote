## 當法院對檢媒共振如何污染審判視而不見，法治國早已質變

**來源：** 光影晨間0630｜2026年4月11日  
**原文連結：** [Facebook 原文](https://www.facebook.com/share/p/1CdqV3Evss/)

> 法治最怕的，不是社會發怒，不是人民上街，不是被告反擊。
> 法治最怕的，是一個握有最終裁判權的體系，已經失去聽見自身噪音的能力。
>
> 當法院在意柯文哲說了什麼，卻刻意忽視檢調外流、媒體先審、輿論定型、辯護權萎縮早已形成的巨大噪音，它捍衛的就不再是完整意義下的審判獨立，而只是對自身權威的選擇性防衛。
>
> 而一個把威權防衛誤當成法治守護的司法，才是真正危險的司法。

---

柯文哲案判決書公布後，其中值得注意的一段，不是枝節性的法律論辯，而是法院在量刑理由中所表現出的那種鮮明態度：案件審理應回到事實認定與法律攻防，而不是受到干擾審判獨立的外部壓力。

這樣的文字，已經不是單純中性敘述，而是明白傳達出一種立場：法院認為，被告及其陣營在法庭外的言行，給法院帶來壓力。

但也正因如此，整件事才顯得格外諷刺。因為最沒有資格裝作自己是審判獨立的純潔守護者的，正是法院。

它對柯文哲與支持者的法庭外發言高度敏感，卻對另一種更長期、更結構性、也更真實存在的審判污染近乎失語：偵查外流、媒體先審、檢媒共振、輿論定型，哪一樣不是一路在法庭外塑造社會心證？如果法院連這些都聽不見，卻只聽得見被告的反擊，那它捍衛的就不是審判獨立，而只是對自身權威的選擇性防衛。

### 一、法院口中的「干擾審判」，為何永遠只朝一個方向看？

法院若要強調審判獨立，當然可以。這本來就是司法存在的基本前提。沒有獨立審判，就沒有值得信賴的裁判；沒有法官不受外力干預的空間，就沒有法治國家最基本的底線。

問題從來不是法院不能講這句話。問題是，法院究竟把什麼事當作干擾審判？

如果被告開記者會、政黨動員支持者、街頭群眾批評司法，這些都足以被法院敏銳感受到，甚至寫進判決理由，視為對審判獨立形成壓力；那麼請問：

- 這些年重大案件中反覆嚴重出現的**偵查資訊外流**，算不算干擾審判？
- 檢方說法總能先一步進入媒體版面，算不算干擾審判？
- 新聞標題、政論節目、社群剪輯在法院尚未完整辯明前，就先把被告定型成貪腐者、權貴者、邪惡者，算不算干擾審判？

如果這些都不算，只有被告與支持者說話才算，那這不叫審判獨立，這叫**選擇性地認定什麼是壓力，什麼不是壓力**。

而一個開始選擇性使用原則的司法，最後耗掉的，不只是它的公信力，而是原則本身。

### 二、真正長期污染審判的，不是被告的反擊，而是檢媒早已完成的法庭外定型

台灣近年重大案件最讓人不安的地方，不是只有判決結果，而是社會的心證，往往早在法庭正式攻防之前，就已經被餵養、定型、分流完畢。

劇本大家都太熟了：

1. 偵查中的片段資訊外流
2. 媒體下重標題、做方向性整理
3. 政論節目與名嘴進行角色安排
4. 社群轉傳、剪輯、二創、情緒放大
5. 在真正判決出來以前，社會上已經形成幾乎不可逆的印象

這整套機制，對審判環境的污染，遠遠比被告一句批評司法的話來得更深、更廣，也更持久。因為它不是短暫的情緒壓力，而是長時間地塑造社會的有罪想像。它讓整個社會在法庭尚未走完之前，就先在情緒上完成審判。到了法院正式宣判的那一天，很多人感受到的，往往已經不是一場完整審理的結束，而只是法庭替法庭外早已定型的故事補上一個制度章。

### 三、法院對被告高度敏感，對檢媒污染卻近乎失語——最刺眼的雙標

法院今天最難看的，不是它談審判獨立，而是它談審判獨立的方式，暴露出一種極其明顯的非對稱：

| 情境 | 法院反應 |
|------|---------|
| 被告發聲 | 立刻覺得受壓迫 |
| 支持者動員 | 立刻覺得審判空間受干擾 |
| 社會批評 | 高舉理性、冷靜、尊重裁判 |
| 偵查外流餵養媒體 | 通常沉默 |
| 媒體先審塑造印象 | 通常寬容 |
| 檢方敘事提前占領輿論 | 通常視若無睹 |

這不是單純的疏忽，而是一種**制度直覺上的偏斜**——對體系內部或與體系關係較近的污染，傾向體諒、合理化、淡化；對被告陣營的批評，則傾向放大、道德化、上綱為對司法威信的威脅。

### 四、法官與檢察官的共生文化，是台灣司法最不願承認的深層病灶

法官與檢察官在制度上角色理應分立，但在實務文化與職業生態裡，兩者之間往往共享太多：共享語言、共享秩序想像、共享面對外部批評時的防衛本能。

於是我們看到的，常常不是徹底拉開距離的制衡關係，而是一種**隱性的共生文化**：

- 檢方若失衡，法院常傾向體諒它背後的偵辦壓力
- 檢方若過度外溢，法院常不願正面處理其對程序環境的影響
- 檢方的說法若已先行占據社會道德高地，法院往往也不願直接承認這對審判氛圍有何種結構性污染

它不是赤裸裸的勾結，更像是一種**制度習慣**：彼此知道彼此的語言，彼此理解彼此的焦慮，彼此在關鍵時刻更傾向守護體系，而不是把刀子往自己體系裡最該被檢討的部分刺進去。

### 五、追訴權對辯護權的侵蝕，正在被正常化

一個健全的刑事訴訟，核心在於被告是否真正擁有**實質受辯護權**——不是紙上說有律師就算數，而是被告能否在沒有寒蟬效應的情況下，放心地與律師討論、準備、建構防禦。

但今天愈來愈多人感受到另一種氛圍：只要案件涉及敏感議題，辯護活動本身就可能被高度可疑化。律師與當事人之間的金流、費用安排、防禦策略，都可能在追訴權的高度敵意視角下被重新解讀。動輒以洗錢、串證、滅證的語言去處理辯護活動邊界，最終造成的，不只是個案壓力，而是**整個辯護制度的收縮**：

- 律師變得更保守
- 被告更不敢充分溝通
- 防禦活動更容易被汙名化
- 最終侵蝕的是所有被告的實質受辯護權

### 六、人民不再尊重司法，不是因為理盲法盲，而是因為看懂了太多

今天很多人不再對司法抱持敬畏，不是因為他們不懂法治，而是因為他們看懂太多事：

- 偵查不公開在很多重大案件中，其實早就不是原則，而是選擇性操作
- 媒體與檢調之間的訊息鏈，足以在法庭攻防前先完成印象管理
- 法院面對這一切時，並沒有拿出絲毫對等程度的自我要求
- 司法體系最擅長的，常常不是清理自身污染，而是在被批評時迅速升高到審判獨立、法治秩序、憲政尊嚴的高度，要求社會自我約束

**尊重從來不是靠要求來的。尊重只能靠一致性、節制、誠實與真正的公正累積而來。**

### 七、真正該被問責的，不是為什麼人民憤怒，而是為什麼司法永遠不先從自己開刀

法院如此不願承認：自己早就不是一個可以只靠法袍、職銜、判決書與新聞稿，就理所當然能獲得尊重的體系。

面對外部批評，它先要求社會尊重。
面對內部污染，它先選擇體諒。
面對程序問題，它先維護制度體面。
面對長期失衡，它先保護自己威信。

這樣的司法，當然還能繼續運作，也當然還能繼續判案。可它已經很難再站在道德高地，質問人民為何不尊重它。

因為真正讓司法失去尊重的，從來不是批評，而是它在最關鍵的地方展現出來的**非對稱、鄉愿與自己人體系維護**。
